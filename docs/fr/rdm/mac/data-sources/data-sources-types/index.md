{{ fr.RDMMAC }} prend en charge plusieurs types de sources de données. Décider d'abord quelle source de données vous allez utiliser.

{% snippet, "badgeNotice" %}
Lors de l'installation initiale, vous utilisez une source de données locale qui est une base de données SQLite.
{% endsnippet %}
