Cet onglet affiche les éléments les plus récemment utilisés/modifiés dans le {{ fr.VLT }} sélectionné. Il est accessible dans le ***{{ fr.NPANE }}***.
![Entrées les plus récemment utilisées](https://cdnweb.devolutions.net/docs/docs_en_rdm_mac_clip10082.png)

Par défaut, dix éléments seront conservés dans l'historique ***Récent***.

L'onglet ***Récent*** peut également être accédé depuis le ***Ruban*** dans ***Affichage - Récent***.
![Affichage - Récent](https://cdnweb.devolutions.net/docs/docs_en_rdm_mac_RDMMac2005.png)
