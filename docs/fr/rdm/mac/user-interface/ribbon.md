Le ***Ruban*** est composé de plusieurs onglets qui contiennent des commandes, classées en sections.

Selon le style d'interface utilisateur choisi, les commandes sont présentées via le ***Ruban*** ou un menu ***Classique***.

![Style Ruban](https://cdnweb.devolutions.net/docs/docs_en_rdm_mac_RDMMac2072.png)

![Style Classique](https://cdnweb.devolutions.net/docs/docs_en_rdm_mac_RDMMac2073.png)

Le style peut être changé dans l'onglet [Compte](/rdm/mac/user-interface/navigation-pane/account/) du ***{{ fr.NPANE }}*** ou dans ***Fichier – Préférences – Interface Utilisateur – Interface Utilisateur***.

![Fichier – Préférences – Interface Utilisateur – Interface Utilisateur](https://cdnweb.devolutions.net/docs/docs_en_rdm_mac_RDMMac2074.png)
