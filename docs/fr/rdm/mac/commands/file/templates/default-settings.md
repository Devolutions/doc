Il est possible d'accéder aux paramètres de session par défaut depuis Fichier - Modèle - Paramètres de session par défaut.

Cela vous permet de modifier ou réinitialiser vos paramètres par défaut qui sont utilisés chaque fois qu'une nouvelle entrée est créée. Chaque type d'entrée est pris en charge et peut avoir un modèle par défaut défini.

En modifiant votre modèle par défaut, vous pourrez sauvegarder les paramètres souvent utilisés lors de la création de nouvelles entrées.

## Paramètres

### Aucun modèle défini

Notez qu'un avis [Aucun par défaut] est affiché sous chaque type qui n'a pas de modèle par défaut défini.
![Modèle - Aucun par défaut](https://cdnweb.devolutions.net/docs/docs_en_rdm_mac_clip10323.png)

### Modèle défini

Tous les types d'entrée sans l'avis [Aucun par défaut] peuvent avoir un modèle par défaut défini pour eux. Vous pouvez double-cliquer sur le type pour modifier le modèle ou appuyer sur le bouton de modification.
![Type - Bouton de modification](https://cdnweb.devolutions.net/docs/docs_en_rdm_mac_clip10324.png)

Si vous souhaitez supprimer le modèle de paramètres par défaut, appuyez sur le bouton de suppression.
![Type - Bouton de suppression](https://cdnweb.devolutions.net/docs/docs_en_rdm_mac_clip10325.png)
