Passer hors ligne se connecte à une copie locale de la source de données lorsque la source de données n'est pas disponible. Cela peut être utilisé lorsqu'un utilisateur travaille à partir d'un réseau déconnecté ou lorsqu'il y a un quelconque problème de connectivité avec la source de données.

{% snippet, "shieldinfo" %}
Il existe des paramètres dans la source de données et dans le compte utilisateur qui peuvent vous empêcher d'utiliser le Mode Hors Ligne. Demander à votre administrateur si le menu n'est pas visible.
{% endsnippet %}

{% snippet, "shieldCaution" %}
Il y a des considérations de sécurité à prendre en compte lors de l'activation du Mode Hors Ligne.
{% endsnippet %}

{{ fr.RDM }} demandera le mode hors ligne lorsque l'application ne peut pas atteindre la source de données.

Activer manuellement le mode hors ligne en utilisant le menu ***Fichier - Passer Hors Ligne***. Lorsque la connexion est de nouveau en ligne, utiliser le menu ***Fichier - Passer En Ligne*** ou l'icône Actualiser pour se reconnecter à la source de données.

Certaines options ne seront pas disponibles en mode Hors Ligne :

* Documents.
* Pièces jointes et journaux.
* [Gestion des utilisateurs](/rdm/mac/commands/administration/user-management/) (Ajouter/Modifier/Supprimer des utilisateurs).

## Mode Hors Ligne

Dans l'onglet Source de Données de Mes Informations sur la Source de Données, le mode hors ligne affiche la taille du fichier hors ligne ainsi que le mode effectif (désactivé, lecture seule ou modifiable). Passer la souris sur le mode hors ligne pour obtenir l'affichage complet du mode hors ligne.
