Ce problème apparaît lorsqu'un utilisateur essaie de mettre à jour ou de désinstaller {{ fr.RDM }}. Désinstaller {{ fr.RDM }} avec le programme et les fonctionnalités de Windows ne fonctionnera pas.
## Solution
La solution est de désinstaller complètement {{ fr.RDM }} en utilisant l'outil Microsoft suivant : [https://support.microsoft.com/en-us/help/17588/fix-problems-that-block-programs-from-being-installed-or-removed](https://support.microsoft.com/en-us/help/17588/fix-problems-that-block-programs-from-being-installed-or-removed)  

Si vous souhaitiez mettre à jour votre instance, vous devrez réinstaller {{ fr.RDM }}. Notre processus de désinstallation ne supprime pas les données personnelles ou la configuration.
