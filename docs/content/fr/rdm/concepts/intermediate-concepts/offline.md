Le mode hors ligne de {{ fr.RDM }} vous permet de vous connecter à une copie locale d'une [source de données](/rdm/windows/concepts/basic-concepts/data-sources/). Les modifications sont enregistrées localement et synchronisées avec la source de données une fois qu'elle est de nouveau en ligne. Les entrées peuvent être accédées même s'il n'y a pas de connexion à la base de données.
