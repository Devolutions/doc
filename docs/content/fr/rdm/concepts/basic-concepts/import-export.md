Importer/exporter des entrées (telles que des sessions, des identifiants, des contacts et des informations d'identification) réduit le temps nécessaire pour configurer vos [{{ fr.VLT }}s](/rdm/windows/concepts/basic-concepts/vaults/), vous permettant ainsi de passer rapidement d'une autre solution à {{ fr.RDM }}. Pour gérer votre [source de données](/rdm/windows/concepts/basic-concepts/data-sources), vous pouvez également exporter un {{ fr.VLT }} entier et son contenu, ainsi qu'une configuration de source de données, puis les réimporter dans une nouvelle source de données ou une instance {{ fr.RDM }}.
