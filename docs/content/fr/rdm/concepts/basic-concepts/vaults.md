Organiser vos [dossiers](/rdm/windows/concepts/basic-concepts/folders/) et entrées dans différents [{{ fr.VLT }}s](/rdm/windows/concepts/basic-concepts/vaults/) au sein d'une [source de données](/rdm/windows/concepts/basic-concepts/data-sources/). Les {{ fr.VLT_MAJ }}s vous permettent de gérer efficacement l'accès en invitant uniquement les utilisateurs qui devraient avoir accès aux données du {{ fr.VLT }}. Définir des {{ fr.VLT }}s par département, client, site physique, etc., pour organiser logiquement vos entrées. Améliorer la [sécurité](/rdm/windows/concepts/advanced-concepts/security/), l'expérience utilisateur de {{ fr.RDM }}, et la performance grâce à l'utilisation de multiples {{ fr.VLT }}s. Avec les {{ fr.UVLT_MAJ }}s, chaque utilisateur dispose de son propre {{ fr.VLT }} sécurisé pour stocker ses entrées uniques.
