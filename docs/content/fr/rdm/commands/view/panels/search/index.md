La fonction ***Rechercher*** dans la section ***Panneaux*** de l'onglet ***Affichage*** permet de rechercher des dossiers ou des entrées dans la base de données sélectionnée.
![Fonction Rechercher](https://cdnweb.devolutions.net/docs/fr/rdm/windows/RdmWin2007.png)

Lors de l'utilisation de la fonction ***Rechercher***, une vue arborescente est également affichée, montrant où se trouvent précisément les résultats de la recherche dans la source de données. En cliquant sur les points de suspension, des options de filtres sont aussi disponibles. Pour en apprendre davantage à leur sujet, consultez [Recherche/Filtre](/fr/rdm/windows/user-interface/status-bar/search-filter/).
![Filtre des sessions](https://cdnweb.devolutions.net/docs/fr/rdm/windows/RdmWin2008.png)

Il est également possible d'effectuer une ***Recherche rapide*** qui permet de chercher dans la base de données sans afficher de vue arborescente. Les résultats sont ainsi plus concis.

{% snippet icon.badgeNotice %}
Truc de pro : Utilisez les raccourcis suivants lorsqu'une entrée est sélectionnée pour rendre votre recherche encore plus rapide.

* Appuyez sur Entrée pour naviguer à l'entrée.
* Appuyez sur <kbd>Ctrl</kbd>+<kbd>Entrée</kbd> pour ouvrir l'entrée.
* Appuyez sur <kbd>Ctrl</kbd>+<kbd>R</kbd> pour naviguer vers le dossier racine de l'entrée.
{% endsnippet %}

![Recherche rapide](https://cdnweb.devolutions.net/docs/fr/rdm/windows/RDMWin2009.png)
