Lorsque vous détenez un grand nombre d'entrées dans votre source de données, il est une meilleure pratique de les compresser pour éviter des problèmes de lenteur lors de l'utilisation de votre source de données. La ***Source de Données (Optimiser)*** analysera toutes vos entrées, les compressera puis les sauvegardera à nouveau, économisant ainsi de l'espace dans votre source de données.

{% snippet, "badgeInfo" %}
Cette fonctionnalité nécessite une [Source de Données Avancée](/rdm/windows/data-sources/data-sources-types/advanced-data-sources/).
{% endsnippet %}

## Paramètres

1. Ouvrir la source de données que vous souhaitez optimiser.
1. Dans le ruban de {{ fr.RDM }}, aller à ***Administration***, et sélectionner ***Source de Données (Optimiser)***.
1. Cliquer sur ***Analyser***.  
![Source de Données (Optimiser) – Analyser](https://cdnweb.devolutions.net/docs/docs_en_rdm_windows_RDMWin6225.png) 
1. Une fois l'analyse terminée, cliquer sur ***Optimiser*** pour procéder à l'optimisation de votre source de données. Vous pouvez sélectionner quel ***{{ fr.VLT }}*** vous souhaitez ***Analyser*** et ***Optimiser***, ou utiliser la fonctionnalité ***Optimiser Tout*** pour effectuer l'action ***Optimiser*** sur tous les ***{{ fr.VLT }}s*** disponibles.
