The ***IP Adress list*** entry keeps the list of IPs in a single entry.

1. In {{ en.RDM }} macOS, select ***Add New Entry***.
1. Choose ***It Asset Management*** – ***IP Address List***.
1. Click ***OK***.
![It Asset Management – IP Address List](https://cdnweb.devolutions.net/docs/docs_en_rdm_mac_RDMMac6050.png) 
1. In the ***Information*** window, enter a ***Name*** and select a ***Folder***.
![Name, IP / Host and Description](https://cdnweb.devolutions.net/docs/docs_en_rdm_mac_RDMMac6051.png)   
1. Go to ***IP Address List*** and click ***Add (+)***.
![IP Address List – Add (+)](https://cdnweb.devolutions.net/docs/docs_en_rdm_mac_RDMMac6052.png)  
1. Enter the ***Name***, ***IP / Host*** and ***Description*** of the IP address.
1. Click ***OK*** to close the window.  
![Enter your IP address](https://cdnweb.devolutions.net/docs/docs_en_rdm_mac_RDMMac6054.png) 

Enter the number of IP addresses to list. You can also edit or delete an IP address.  

8. Click ***Save*** to save the entry.  
![Enter the number of IP addresses you want to list](https://cdnweb.devolutions.net/docs/docs_en_rdm_mac_RDMMac6053.png)   
The ***IP Adress list*** entry is saved.


![Enter your IP addresses](https://cdnweb.devolutions.net/docs/docs_en_rdm_mac_RDMMac6055.png)
