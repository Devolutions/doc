Un compte à ***privilèges nuls permanents*** (ZSP) dans Devolutions PAM n'a aucune appartenance à un groupe ou permissions attachées. Si compromis, un attaquant ne pourra pas obtenir davantage de contrôle. Les comptes ZSP réduisent considérablement l'exposition, car ils maintiennent les privilèges pour de courtes durées.
