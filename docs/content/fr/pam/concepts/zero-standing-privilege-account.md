Un compte à ***privilèges nuls permanents*** (ZSP) dans Devolutions PAM est un compte qui ne possède aucune permission et n'est membre d'aucun groupe. Si compromis, il devrait être impossible pour l'attaquant d'accomplir quoi que ce soit. Cela réduit considérablement l'exposition, car le compte n'est utilisé que pour une petite fraction du total des heures dans une année. Chez Devolutions, les comptes privilégiés (PAs) sont typiquement réservés pour quelques heures par semaine, minimisant davantage le risque.
