[La Gestion des Accès Privilégiés de Devolutions](https://devolutions.net/privileged-access-management/) pour {{ fr.DHUBB }} prend en charge les comptes privilégiés Azure AD. Elle est spécialement conçue pour répondre aux besoins des PME, offrant des fonctionnalités de niveau entreprise pour apporter un niveau de protection habituellement réservé aux grandes organisations tout en étant robuste, facile à déployer et abordable.

Voici certaines de ses principales fonctionnalités :

* Coffre PAM sécurisé gérant tous vos différents comptes privilégiés
* Approbation de demande de réquisition
* Réinitialisation automatique du mot de passe
* Injection sécurisée de mot de passe
* Rapport d'administration

Puisque {{ fr.DHUBB }} est une solution basée sur le nuage, vous devez télécharger et installer le [{{ fr.DHUBS }}](/pam/hub/download-and-install-devolutions-hub-services/) pour établir la communication entre votre {{ fr.DHUBB }} et vos ressources internes.
