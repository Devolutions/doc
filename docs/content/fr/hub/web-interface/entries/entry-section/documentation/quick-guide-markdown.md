Markdown est un langage de balisage léger utilisé pour la ***Documentation*** dans les entrées.

Pour votre commodité, nous avons ajouté des boutons de formatage avec des polices, des styles et des insertions. Pour l'utiliser, cliquer sur un bouton de formatage et insérer vos données.
![!!Hub4083](https://cdnweb.devolutions.net/docs/docs_en_hub_Hub4083.png)
{% snippet, "badgeNotice" %}
Cliquer sur les boutons ***Aperçu*** ou ***Diviser verticalement*** pour revoir le format du document.
{% endsnippet %}

### Titres Markdown Manuel

Il existe deux manières de créer des titres manuels avec Markdown.

Les premier et deuxième niveaux peuvent être créés en « soulignant » le texte avec des signes égaux (=) et des tirets (-).
