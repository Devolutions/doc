La fonctionnalité ***Accès Temporaire*** permet aux utilisateurs de demander une élévation temporaire de leurs permissions pour une entrée spécifique. Les approbateurs peuvent octroyer un ***Accès Temporaire*** sans demande de la part d'un utilisateur et même si l'utilisateur a déjà accès.

## Octroyer un accès temporaire

1. Sélectionner l'entrée à laquelle octroyer l'accès.
1. Cliquer sur ***Plus***, puis sélectionner ***Octroyer un Accès Temporaire***.
![Octroyer un Accès Temporaire](https://cdnweb.devolutions.net/docs/docs_en_hub_Hub6033.png)
Vous pouvez voir les ***entrées liées*** telles que les identifiants liés et les VPN.
![Fenêtre Octroyer un Accès Temporaire](https://cdnweb.devolutions.net/docs/docs_en_hub_Hub6021.png)
1. Choisir la ***Durée d'accès*** dans le menu déroulant.
![Durée d'accès](https://cdnweb.devolutions.net/docs/docs_en_hub_Hub6015.png)
Vous pouvez également cliquer sur ***Définir une période personnalisée*** si vous avez besoin d'octroyer l'accès pour une période spécifique.
![Définir une période personnalisée](https://cdnweb.devolutions.net/docs/docs_en_hub_Hub6016.png)
1. Sélectionner le niveau de ***Permissions*** que vous souhaitez obtenir temporairement entre ***Lecteurs***, ***Opérateurs***, et ***Contributeurs*** en utilisant le menu déroulant.
![Définir une période personnalisée](https://cdnweb.devolutions.net/docs/docs_en_hub_Hub6017.png)
1. Passer votre souris sur l'icône œil pendant que vous avez un niveau de ***Permissions*** sélectionné vous permet de voir quelles permissions il inclut. Par exemple, l'image suivante montre les permissions pour le rôle ***Opérateurs***.
![Contenu des Permissions](https://cdnweb.devolutions.net/docs/docs_en_hub_Hub6018.png)
Pour vous aider à décider quel rôle sélectionner, vous pouvez visiter [Rôles et permissions](https://docs.devolutions.net/hub/web-interface/administration/configuration-security/system-permissions/roles-permissions/).
1. Sélectionner l'***Utilisateur***. Vous pouvez également utiliser la barre de filtre pour affiner votre recherche.
![Utilisateur](https://cdnweb.devolutions.net/docs/docs_en_hub_Hub6019.png)
Un court message peut être laissé à l'***Utilisateur*** expliquant pourquoi vous octroyez l'accès.
![Message](https://cdnweb.devolutions.net/docs/docs_en_hub_Hub6020.png)
1. Cliquer sur ***Octroyer l'Accès*** pour fermer la fenêtre.
