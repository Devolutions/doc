{% snippet, "badgeInfo" %}
Commencez votre essai gratuit de {{ fr.DHUBB }} [aujourd'hui](https://password.devolutions.net/business/sign-up) !
{% endsnippet %}

1. Accéder à la section [{{ fr.DPORTAL }}](https://portal.devolutions.com/password-hub) de votre {{ fr.DA }} pour créer votre {{ fr.DHUBB }}.
1. Cliquer sur ***{{ fr.DHUBB }}*** dans le menu de gauche.
1. Remplir votre ***nom d'entreprise*** dans la section d'essai gratuit de 30 jours.
1. Choisir un nom de site ***{{ fr.DHUB }}***.
{% snippet, "badgeCaution" %}
Le nom du site ***{{ fr.DHUB }}*** devient votre adresse de site Web et ne peut pas être modifié après le déploiement de votre hub.
{% endsnippet %}

5. Sélectionner la région où vous souhaitez que vos données soient [hébergées](/hub/kb/hub-business/knowledge-base/hosting-region-password-hub/) et accepter les ***Conditions des services en ligne et la Politique de confidentialité***.
![Commencer l'essai](https://cdnweb.devolutions.net/docs/docs_en_hub_Hub4009.png)
1. Cliquer sur ***Commencer l'essai***.

Une fois le déploiement prêt, vous recevrez un courriel avec l'adresse de votre site Web {{ fr.DHUBB }}.

{% snippet, "badgeHelp" %}
Si vous êtes un utilisateur de l'édition Team de {{ fr.RDM }}, vous pouvez connecter votre {{ fr.DHUBB }} comme source de données dans ***Fichier – Sources de données – Ajouter une nouvelle source de données***. En savoir plus sur [l'intégration de {{ fr.DHUBB }} dans {{ fr.RDM }}](/rdm/kb/rdm-windows/how-to-articles/integrate-hub-business-rdm/).
{% endsnippet %}

7. Cliquer sur l'adresse du site Web {{ fr.DHUBB }}.
1. Télécharger votre ***Kit d'urgence***.
![Kit d'urgence](https://cdnweb.devolutions.net/docs/docs_en_hub_Hub4170.png)

{% snippet, "badgeCaution" %}
Le ***Kit d'urgence*** contient votre clé d'urgence qui sera nécessaire en cas de perte ou de réinitialisation de votre mot de passe {{ fr.DA }}. N'oubliez pas de le conserver dans un endroit sûr.
{% endsnippet %}

Après avoir téléchargé le ***Kit d'urgence***, vous serez dirigé vers votre {{ fr.DHUBB }}. Pour apprendre comment récupérer l'accès à votre {{ fr.DA }} avec le ***Kit d'urgence***, cliquez [ici](/hub/kb/general-knowledge/emergency-kit/).
