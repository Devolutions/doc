Les {{ fr.DCS }} sont le lieu central pour tous les services en ligne de Devolutions. Ils comprennent plusieurs services comme résumé ci-dessous.

La caractéristique commune entre tous ces services est qu'ils ne nécessitent que la possession d'un {{ fr.DA }} gratuit pour être accessibles.
