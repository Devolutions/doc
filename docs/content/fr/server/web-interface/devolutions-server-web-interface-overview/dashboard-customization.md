{{ fr.DVLS }} les widgets du tableau de bord peuvent être ajoutés ou supprimés en cliquant sur ***Personnaliser les Widgets***.

![Bouton de personnalisation du tableau de bord](https://cdnweb.devolutions.net/docs/DVLS6067_2024_1.png)

Dans la fenêtre ***Personnaliser les Widgets***, la visibilité des widgets peut être définie sur ***Toujours, Jamais*** ou ***Si non vide***.

![La visibilité des widgets peut être définie sur Toujours, Jamais ou Si non vide](https://cdnweb.devolutions.net/docs/DVLS6068_2024_1.png)
