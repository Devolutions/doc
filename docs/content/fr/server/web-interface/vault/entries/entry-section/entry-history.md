Consulter, restaurer, comparer, supprimer et exporter l'***Historique des Entrées*** des entrées de {{ fr.DVLS }}.
### Restaurer
Pour restaurer une version précédente d'une entrée spécifique, cliquer sur le bouton ***Restaurer***.
![Bouton Restaurer](https://cdnweb.devolutions.net/docs/docs_en_server_ServerOp4004.png)
### Comparer
Pour comparer les versions des entrées, sélectionner deux entrées et cliquer sur le bouton ***Comparer***.
![Bouton Comparer](https://cdnweb.devolutions.net/docs/docs_en_server_ServerOp4005.png)
La fenêtre ***Comparer les Modifications de Session*** montre les changements entre les entrées sélectionnées.
![Fenêtre Comparer les Modifications de Session](https://cdnweb.devolutions.net/docs/docs_en_server_ServerOp0000.png)
### Supprimer
Pour supprimer une entrée, appuyer sur le bouton ***Supprimer***.
![Bouton Supprimer](https://cdnweb.devolutions.net/docs/docs_en_server_ServerOp0001.png)
### Exporter
Pour sauvegarder l'***Historique des Entrées***, cliquer sur ***Exporter***. Un fichier .csv sera téléchargé.
![Bouton Exporter](https://cdnweb.devolutions.net/docs/docs_en_server_ServerOp0002.png)
