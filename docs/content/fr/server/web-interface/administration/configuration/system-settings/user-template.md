Cette section définira le modèle d'utilisateur pour la fonctionnalité [Création automatique d'utilisateur](/server/web-interface/administration/configuration/server-settings/general/authentication/domain/).
![Administration – Paramètres système – Modèle d'utilisateur](https://cdnweb.devolutions.net/docs/docs_en_server_ServerOp8033.png)

## Modèle d'utilisateur
| Option            | Description                         |
|-------------------|-------------------------------------|
| Mode hors ligne   | Cette option affectera uniquement l'application {{ fr.RDM }}. <br>Les valeurs possibles sont : <ul><li>***Désactivé***</li><li>***Lecture seule***</li><li>***Lecture/écriture***</li></ul> |
| Type d'utilisateur| Sélectionner le type d'utilisateur. <br>Les valeurs possibles sont : <ul><li>***Administrateur***</li><li>***Utilisateur***</li><li>***Utilisateur en lecture seule***</li></ul> |
| Type de licence utilisateur | Sélectionner le type de licence que l'utilisateur possède : <br><ul><li>***Par défaut***</li><li>***Gestion de connexion***</li><li>***Gestion de mot de passe***</li></ul> |
| Langue            | Sélectionner la langue par défaut de l'utilisateur. |
