Suivre ces étapes pour connecter {{ fr.RDMWIN }} au {{ fr.WBEX }}. Vous serez invité à associer les deux applications.

1. Cliquer sur le {{ fr.WBEX }} en haut à droite de votre navigateur.
   {% snippet, "badgeCaution" %}
   {{ fr.RDM }} doit être installé et en cours d'exécution pour continuer.
   {% endsnippet %}

1. Choisir ***{{ fr.RDM }}*** dans la liste, puis cliquer ***Enregistrer***.
![Première Connexion](https://cdnweb.devolutions.net/docs/docs_en_rdm_windows_Dwl4056.png)
1. Entrer un nom pour cette association dans la zone de texte.
![Association avec {{ fr.RDM }}](https://cdnweb.devolutions.net/docs/docs_en_rdm_windows_Dwl4057.png)

1. Cliquer sur ***Associer avec {{ fr.RDM }}***.
   {% snippet, "badgeInfo" %}
   Une fenêtre pop-up apparaîtra dans {{ fr.RDM }} pour confirmer que la demande a bien été envoyée par vous.
   {% endsnippet %}

1. Cliquer ***Oui*** dans {{ fr.RDM }} pour accepter la demande d'association.
![Demande d'Association](https://cdnweb.devolutions.net/docs/docs_en_rdm_windows_Dwl4058.png)
1. Si vous cliquez sur l'icône dans votre navigateur web, vous pouvez valider que le {{ fr.WBEX }} est maintenant associé à votre {{ fr.RDM }}.
![Association Réussie de {{ fr.RDM }} au {{ fr.WBEX }}](https://cdnweb.devolutions.net/docs/docs_en_rdm_windows_Dwl4059.png)
   {% snippet, "badgeInfo" %}
   Notez que, bien que {{ fr.RDM }} puisse gérer plusieurs associations (par exemple, plusieurs navigateurs), le {{ fr.WBEX }} ne peut être associé qu'à une seule instance de {{ fr.RDM }} à la fois.
   {% endsnippet %}
