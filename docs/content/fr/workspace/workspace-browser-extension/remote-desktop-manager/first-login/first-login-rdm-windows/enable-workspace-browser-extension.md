Le {{ fr.WBEX }} est activé par défaut dans les propriétés d'entrée du {{ fr.RDMWIN }}. S'il est désactivé, vous pouvez le réactiver en sélectionnant ***Par défaut (Activé)*** ou ***Activé*** dans la liste déroulante du {{ fr.WBEX }}.
