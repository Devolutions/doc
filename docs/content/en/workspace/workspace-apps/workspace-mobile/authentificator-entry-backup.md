Cloud backup settings are available to backup all entries using an ***Authenticator entry backup*** with {{ en.DHUBP }}. 
1. Open {{ en.WMAPP }}.
1. To enable ***Cloud backup***, select ***Authenticator*** under ***Active Spaces***.
![Active Spaces section](https://cdnweb.devolutions.net/docs/docs_en_hub_Hub6088.png)
1. Tap ***Cloud Backup options***.
![Cloud Backup options](https://cdnweb.devolutions.net/docs/docs_en_hub_Hub6089.png)
1. Enable ***Use cloud backup***.
![Cloud Backup options](https://cdnweb.devolutions.net/docs/docs_en_hub_Hub6087.png)

| OPTION                    | DESCRIPTION |
|---------------------------|-------------|
| Currently selected account| Show the ***Currently selected account.*** The account can be changed. |
| Sync only with this device| Re-sync all MFA and SSO entries with this device only. |
