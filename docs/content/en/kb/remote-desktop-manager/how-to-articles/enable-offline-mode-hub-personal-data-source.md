In {{ en.RDM }}, to use the Offline Mode with a {{ en.DHUBP }} data source, you must first enable the feature in the {{ en.DHUBP }} web interface settings, as it is disabled by default.

## Steps

1. In {{ en.DHUBP }}, go to your account, then click on ***Preferences***.
![Account â€“ Preferences](https://cdnweb.devolutions.net/docs/RDMW4019_2024_1.png)
1. In ***General***, under ***Offline settings***, click to ***Allow offline***.
![Allow offline mode](https://cdnweb.devolutions.net/docs/RDMW4017_2024_1.png)
1. Click ***Close***.
