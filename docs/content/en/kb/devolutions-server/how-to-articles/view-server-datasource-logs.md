{% endsnippet %}

The {{ en.DVLS }} logs can be consulted in Reports - Data Source Logs

![!!KB8011](https://cdnweb.devolutions.net/docs/en/kb/KB8011.png)

Or it can be viewed from the {{ en.DVLSCONSOLE }} using the Logs button.
