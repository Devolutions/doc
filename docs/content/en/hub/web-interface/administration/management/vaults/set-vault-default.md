You can set a specific {{ en.VLT }} to open by default when launching your {{ en.DHUBB }}.

1. Select the {{ en.VLT }} to set as default.
1. In the ***Filter*** box, click ***Display Settings***.
1. Enable ***Set current {{ en.VLT }} as default***.
