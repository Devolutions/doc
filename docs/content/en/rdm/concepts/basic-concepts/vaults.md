Organize your [folders](/rdm/windows/concepts/basic-concepts/folders/) and entries in different [{{ en.VLT }}s](/rdm/windows/concepts/basic-concepts/vaults/) within a [data source](/rdm/windows/concepts/basic-concepts/data-sources/). {{ en.VLT_MAJ }}s allow you to efficiently manage access by inviting only the users who should have access to the {{ en.VLT }} data. Define {{ en.VLT }}s by department, client, physical site, etc., to logically organize your entries. Improve the [security](/rdm/windows/concepts/advanced-concepts/security/), user experience of {{ en.RDM }}, and performance through the use of multiple {{ en.VLT }}s. With {{ en.UVLT }}s, each user has their own secure {{ en.VLT }} to store their unique entries.
