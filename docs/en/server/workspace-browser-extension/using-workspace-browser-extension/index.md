{% snippet icon.badgeNotice %} 
This topic assumes that you have previously [installed the {{ en.WBEX }} extension](/server/workspace-browser-extension/installation/) in your browser and that it is [connected to {{ en.DVLS }}](/server/workspace-browser-extension/first-login-workspace-browser-extension/). 
{% endsnippet %}
 
The {{ en.WBEX }} is the tool of choice for users to interact in a password management context with {{ en.DVLS }}. It allows for:  

* automatically matching websites to their respective saved credentials. 
* directly viewing and filtering through the accounts that their {{ en.VLT }}s contain. 
* updating and even adding new websites and entries. 
* quickly accessing a password generator when registering new websites. 
* automatically detecting when new credentials are manually filled for a website and saving them.  

Consult these different topics to learn about useful {{ en.WBEX }} features:  

* [Create an Account for a Website](/server/workspace-browser-extension/using-workspace-browser-extension/create-account-website-workspace-browser-extension/)  
* [Add a Website Entry](/server/workspace-browser-extension/using-workspace-browser-extension/add-website-entry-workspace-browser-extension/)  
* [Retrieve Credentials](/server/workspace-browser-extension/using-workspace-browser-extension/retrieve-credentials-workspace-browser-extension/)  
* [Keyboard Shortcuts](/server/workspace-browser-extension/settings/keyboard-shortcuts/)
