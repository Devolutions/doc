  title: Device Login
---
The ***Device Login*** section is to set up the Cloud backup of your {{ en.DWS }} entries in {{ en.DHUBP }}.  

When unlocked, go to your {{ en.DWS }} app and create a backup. This will allow you to restore previous backup.
