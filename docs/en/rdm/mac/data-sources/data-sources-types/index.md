{{ en.RDMMAC }} supports multiple types of data source. First decide which data source you are going to use.  

{% snippet icon.badgeNotice %} 
Upon initial installation, you will be running from a local data source which is an SQLite database. 
{% endsnippet %}
 
## Data Source Types 

| ICON | NAME | DESCRIPTION | PROS AND CONS |
|------|------|-------------|---------------|
| ![!!DataSourceDODLarge](https://webdevolutions.azureedge.net/docs/common/DataSourceDODLarge.png) | {{ en.DOD }} | {{ en.RDMMAC }} uses {{ en.DOD }} to store and synchronize your sessions. Access your sessions from anywhere using an Internet connection. For more information, please consult [{{ en.DOD }}](/cloud/rdm-online-services/online-drive/). | Pros: <ul><li>Quick.</li><li>Reliable.</li><li>Free service.</li></ul> Cons: <ul><li>No possibility for sharing.</li><li>No security management.</li></ul> |
| ![!!dvls-90X90](https://webdevolutions.azureedge.net/docs/common/dvls-90X90.png) | {{ en.DVLS }} | {{ en.RDMMAC }} uses {{ en.DVLS }} to store session information. For more information, please consult [{{ en.DVLS }}](/rdm/mac/data-sources/data-sources-types/advanced-data-sources/server/). | Pros: <ul><li>Quick.</li><li>Reliable.</li><li>Secure.</li><li>Supports all features, such as attachments, Usage Logs, Offline Mode and User Management.</li><li>Active Directory integration.</li></ul> Cons: <ul><li>Installation required.</li></ul> |
| ![!!DataSourceDropBoxLarge](https://webdevolutions.azureedge.net/docs/common/DataSourceDropBoxLarge.png) | Dropbox | {{ en.RDMMAC }} uses the Dropbox API to retrieve the XML file from the configured repository. For more information, please consult [Dropbox](/rdm/mac/data-sources/data-sources-types/dropbox/). | Pros: <ul><li>Can be shared in read-only mode.</li><li>Backups (by Dropbox) are automatic.</li><li>Storage infrastructure is free (if within your free storage quota).</li></ul> Cons: <ul><li>No security management.</li><li>There is a possibility for conflict or data corruption to occur.</li><li>Does not support all features, such as attachments, Usage Logs, Offline Mode and User Management.</li><li>Dropbox integration uses the Dropbox SDK, so features exclusive to the Business or Enterprise editions are NOT supported.</li></ul> |
| ![!!DataSourceAzureLarge](https://webdevolutions.azureedge.net/docs/common/DataSourceAzureLarge.png) | Microsoft Azure SQL | {{ en.RDMMAC }} uses the Microsoft cloud platform to save and manage all sessions. For more information, please consult [Microsoft Azure SQL](/rdm/mac/data-sources/data-sources-types/advanced-data-sources/microsoft-azure-sql/). | Pros: <ul><li>Quick.</li><li>Reliable.</li><li>Secure.</li><li>Supports all features, such as attachments, Usage Logs, Offline Mode and User Management.</li></ul> Cons: <ul><li>Microsoft Azure needs to be configured.</li></ul> |
| ![!!DataSourceSQLServerLarge](https://webdevolutions.azureedge.net/docs/common/DataSourceSQLServerLarge.png) | Microsoft SQL Server | {{ en.RDMMAC }} uses SQL Server to save and manage all sessions. This is one of the available data source for a multi-user environment. For more information, please consult [SQL Server (MSSQL)](/rdm/mac/data-sources/data-sources-types/advanced-data-sources/microsoft-sql-server/). | Pros: <ul><li>Quick.</li><li>Reliable.</li><li>Secure.</li><li>Supports all features, such as attachments, Usage Logs, Offline Mode and User Management.</li><li>SQLServer Express is free.</li></ul> Cons: <ul><li>SQLServer must be installed.</li></ul> |
| ![!!ApplicationHubLarge](https://webdevolutions.azureedge.net/docs/common/ApplicationHubLarge.png) | {{ en.DHUBB }} | {{ en.RDMMAC }} connects to the {{ en.DHUBB }} {{ en.VLT }}. Note that there are different subscription levels for this product. For more information, please see [Products features and highlights](https://devolutions.net/password-hub) and consult [{{ en.DHUBB }}](/rdm/mac/data-sources/data-sources-types/advanced-data-sources/hub-business/). | Pros: <ul><li>Quick.</li><li>Reliable.</li><li>Secure.</li><li>Shareable.</li></ul> Cons: <ul><li>Cannot be self-hosted.</li><li>No offline mode.</li></ul> |
| ![!!Hub-personal-32x32](https://webdevolutions.azureedge.net/docs/common/hub-personal-32x32.png) | {{ en.DHUBP }} | {{ en.RDMMAC }} uses {{ en.DHUBP }} to store and synchronize your sessions. Access your sessions from anywhere using an Internet connection. For more information, please consult [{{ en.DHUBP }}](/rdm/mac/data-sources/data-sources-types/hub-personal/). | Pros: <ul><li>Quick.</li><li>Reliable.</li><li>Free service.</li></ul> Cons: <ul><li>No possibility for sharing.</li><li>No offline mode.</li></ul> |
| ![!!DataSourceSQLiteLarge](https://webdevolutions.azureedge.net/docs/common/DataSourceSQLiteLarge.png) | SQLite | {{ en.RDMMAC }} uses a SQLite database to store session information. For more information, please consult [SQLite](/rdm/mac/data-sources/data-sources-types/sqlite/). | Pros: <ul><li>Quick.</li><li>Reliable.</li><li>Free database.</li><li>Supports all features, such as attachments and Usage Logs, Offline Mode and User Management.</li></ul> Cons: <ul><li>No possibility for sharing.</li><li>No security management.</li></ul> |
| ![!!DataSourceXMLLarge](https://webdevolutions.azureedge.net/docs/common/DataSourceXMLLarge.png) | XML | {{ en.RDMMAC }} saves the settings directly in a file with the XML format.
